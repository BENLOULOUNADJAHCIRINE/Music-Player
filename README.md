# 🎵 Music Player

A clean, minimal music player built with HTML, CSS, and JavaScript.

## Preview

> A single-song music player with a sleek dark UI, animated album art border, and full playback controls.

## Features

- ▶️ Play / Pause toggle
- ⏪ Rewind 10 seconds
- ⏩ Fast Forward 10 seconds
- 🔁 Repeat song
- 📊 Progress bar with drag to seek
- ⏱️ Live current time display
- 🎨 Animated gradient border on album art
- 🔊 Auto-repeat when song ends

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- HTML5 Audio API
